# Student Performance Analysis Dashboard | Power BI

## Project Overview

This project is a **Student Performance Analysis Dashboard** built using **Microsoft Power BI**.  
The dashboard provides insights into student academic performance by analyzing scores, demographics, test preparation, and parental education factors.

The goal of this project is to transform raw student data into an interactive report that helps understand performance patterns and key factors affecting student outcomes.

---

## Tools & Technologies Used

- Microsoft Power BI
- Power Query
- DAX (Data Analysis Expressions)
- Data Visualization
- Excel Dataset

---

## Dataset Description

The dataset contains student-related information such as:

- Student ID
- Gender
- Ethnicity
- Parental Education Level
- Test Preparation Status
- Math Score
- Reading Score
- Writing Score

The dataset was used to analyze academic performance trends and identify relationships between different factors.

---

## Data Cleaning & Transformation (Power Query)

The data was prepared using Power Query before building the dashboard.

Steps performed:

- Removed duplicate records
- Handled missing/null values
- Changed incorrect data types
- Cleaned and formatted columns
- Prepared data for analysis

---

## Dashboard Features

### Key Performance Indicators (KPIs)

The dashboard displays:

- Total Student Count
- Average Math Score
- Average Reading Score
- Average Writing Score
- Average Overall Score Percentage

---

## Visual Analysis

The dashboard includes:

### 1. Performance by Ethnicity
- Compares average student scores across different ethnic groups

### 2. Performance by Gender
- Analyzes score distribution between male and female students

### 3. Parental Education Impact
- Shows how parental education level influences student performance

### 4. Test Preparation Analysis
- Compares performance of students who completed test preparation versus those who did not

### 5. Top Performing Students
- Displays students with the highest performance scores
